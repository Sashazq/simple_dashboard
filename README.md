# App Dashboard

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Architecture

This project follows a **feature-based design** pattern, organizing code by feature modules:

- `features/` — Feature-specific components, hooks, context, and utilities
- `shared/` — Shared components and utilities used across features
- `(dashboard)/` — Dashboard route group with layout

## Getting Started

First, run the development server:

```bash
npm ci
npm run dev
```
